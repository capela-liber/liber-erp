Arquivos no Google Drive

O mesmo porteiro do Dropbox, agora na frente do Drive: leitura e escrita decididas pasta a pasta no Odoo, download conferido byte a byte, compartilhamento com prazo e assinatura — para a equipe que vive na planilha e no documento do Google.

Visão geral

Este app é um corpo do chassi Cloud Files — o mesmo do Dropbox e do GitHub. Toda a disciplina é herdada e idêntica: pastas mapeadas por empresa, grupos de Leitura e Escrita por pasta, espelho só de metadados, envio que nunca sobrescreve, tags, vínculos com contatos e produtos, e a regra de ouro — mapear pastas é de administrador; escrever exige grupo de escrita, seja quem for. O que muda aqui é só o jeito do Drive:

  • Pasta é ID, não caminho. Abra a pasta no navegador e copie o ID da URL (drive.google.com/drive/folders/<ID>) para o campo ID externo do mapeamento.
  • O download passa pelo Odoo. O Drive não tem link anônimo de quatro horas; o arquivo desce pela porta /liber_cloud/download, com o portão conferido antes do primeiro byte — a credencial nunca sai do servidor.
  • Miniatura de quase tudo, inclusive PDF — aqui o Drive é mais generoso que o Dropbox.

Configuração

  1. No console.cloud.google.com, crie um projeto, ative a Google Drive API e gere uma credencial OAuth (tipo Desktop app). O passo a passo com o comando que troca o código pelo refresh token está no NOTES.md do módulo.
  2. Preencha Drive‣Configuração‣Conta — Client ID, Client Secret, Refresh Token — e use Testar conexão. Uma conta por empresa.
  3. Como administrador, mapeie as pastas em Drive‣Configuração‣Pastas: nome, o ID externo da pasta no Drive, grupos de leitura e escrita.
Atenção

Expiração de link compartilhado no Drive é recurso de Google Workspace pago. Em conta gratuita a criação do link com prazo é recusada com o erro explicado — zere o prazo na Conta ou faça upgrade.

O dia a dia

Igual ao Dropbox, de propósito: sincronizar (botão, após cada envio, e o robô diário de madrugada), baixar, enviar pelo menu Drive‣Enviar, compartilhar com confirmação e prazo, classificar com tags e vincular a contatos e títulos — o vocabulário de tags é o mesmo da casa inteira, e os botões nas fichas de contato e produto somam os arquivos de todas as estantes.

Limitações conhecidas

  • Arquivos nativos do Google (Docs, Sheets) aparecem no espelho, mas o download cru deles depende de exportação — fase futura; abra-os pelo link compartilhado.
  • Expiração de link exige Workspace pago.
  • Drives compartilhados (Shared Drives) ainda não são varridos — mapeie pastas de "Meu Drive" por enquanto.
